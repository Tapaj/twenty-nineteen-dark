# Twenty Nineteen Dark

A child theme of Twenty Nineteen  
This is just a test theme for created learning purpose.

## Pre-requisite

The Twenty Nineteen theme should be installed in your WordPress installation.  
If you don't have it installed, you can download it from [WordPress's Theme Directory](http://https://wordpress.org/themes/twentynineteen/)

## Installation

Run the following command inside the theme folder of your WordPress Installation:

```bash
git clone https://github.com/Tapaj/twenty-nineteen-dark.git
```

## Changes

- Colour scheme

- Customized 404 Page
- Modified Author Byline in Posts
